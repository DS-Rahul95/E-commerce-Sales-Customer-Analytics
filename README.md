# E-commerce Sales & Customer Analytics

## Objective
Analyze e-commerce sales data to identify revenue drivers, customer behavior, and profitability trends.

## Tools Used
- SQL (MySQL)
- Power BI
- Excel

## Key Insights
- Top 20% customers contributed ~60% revenue
- High discounts (>20%) reduced overall profitability
- Technology category generated highest sales
- Repeat customers drove significant revenue

## Dashboard Features
- Sales & Profit KPI tracking
- Customer segmentation analysis
- Product performance insights
- Regional sales distribution

## Files Included
- SQL Queries
- Power BI Dashboard
- Dataset
